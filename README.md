# 📱 WELLIO – STUDENT WELLBEING & PRODUCTIVITY APP

## 📖 OVERVIEW
WELLIO is a mobile application designed to help students manage their academic workload while monitoring their emotional wellbeing.

The system integrates:
- Academic scheduling
- Mood tracking
- Data analysis and visualisation
- Stress detection and alert mechanisms

The main objective is to provide students with early warnings of stress and actionable insights to maintain a healthy balance between studies and wellbeing.

---

## 🚀 KEY FEATURES
- Secure login using Firebase Authentication  
- Google Calendar integration for automatic deadline tracking  
- Mood logging with notes and tags  
- Visual dashboard displaying trends (mood, sleep, workload)  
- AI-based stress detection and analysis  
- Notification and alert system  

---

## 🛠️ TECHNOLOGY STACK
- Frontend: Mobile Application UI  
- Backend: API Services  
- Database: Firebase  
- Authentication: Firebase Authentication  
- Integration: Google Calendar API
